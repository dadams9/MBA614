Working with Files

One common way to share data between systems in through text files. Sometimes the format of a text file you may need to process will one that Excel knows how to import directly, other times, you many need to work directly with the file through VBA. In this exercise you will read the first line of a text file into a module-level variable.

Task #PointsTask Description
1 10 In the workbook named "ReadTextFile-XX.xlsm," you will find a module named “Solution” with a sub procedure named "ReadFirstLine”.  Your assignment is to complete the ReadFirstLine sub procedure so that when it is executed, it will read the first line from a text file named "report1.txt" and write its value to cell A1 of the active sheet.

The “report1.txt” text file should be located in the same folder as the workbook in which you are writing your code.  It is generated the first time you open these instructions.  The following expression will return the full path to the “report1.txt” file:

ThisWorkbook.path & Application.pathSeparator & "report1.txt"
